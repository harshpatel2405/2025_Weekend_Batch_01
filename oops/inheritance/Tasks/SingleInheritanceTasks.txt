QUESTION 2: EMPLOYEE AND MANAGER
=========================================

Create a base class Employee with the following data members:

- employeeId
- employeeName
- monthlySalary

- department

Name          : Krishna

Monthly Salary: 80000

QUESTION 4: SHAPE AND RECTANGLE
=========================================

Create a base class Shape with the following data members:

- length
- breadth

Create a derived class Rectangle that inherits from Shape.

Requirements:
1. Accept length and breadth from the user.
2. Calculate Area of the Rectangle.
3. Calculate Perimeter of the Rectangle.
4. Display all results.
5. Use Single Inheritance.

Formulas:

Area = Length × Breadth

Perimeter = 2 × (Length + Breadth)

Sample Output:

----- Rectangle Details -----
Length    : 10
Breadth   : 5

Area      : 50
Perimeter : 30
